Paragrid Level Editor

Import Grids

Paste grid definitions in JSON5 format (unquoted keys OK). Example: 
{
  main: "1 2|3 4",
  grid_2: "*main 5|6 7"
}
Format: Object where keys are grid IDs and values are grid strings. Use | to separate rows, spaces for cells. Digits = concrete cells, letters = refs, * = primary ref, ~ = secondary ref, _ = empty.
